# 東京メトロ (メトポ / Metpo)

東京メトロのポイントサービス「メトポ」の認証および取得方法について解説します。

## 認証方法

### 必要な情報
- **お客様番号 (Customer Number)**: 10桁の数字
- **WEBパスワード (Web Password)**: 登録時に設定したパスワード

### 手順
1. [メトポ公式サイト](https://www.metro-point-club.jp/)で会員登録を行います。
2. 多機能券売機等でICカード（PASMO）の登録を完了させます。
3. お客様番号とパスワードを環境変数 `TOKYO_METRO_EMAIL` (お客様番号) と `TOKYO_METRO_PASSWORD` に設定します。

## ポイント取得の仕組み

### 技術的詳細
- **認証**: `https://www.metro-point-club.jp/mp/contents/wb1004/ws0010101/login` に対して `POST` リクエストを送信し、セッションCookieを取得します。
- **データ取得**: ログイン後のトップページ (`wb1005/ws0030101/view`) を `GET` し、HTMLをパース（スクレイピング）します。
- **取得項目**:
    - 保有ポイント（合計）
    - 通常ポイント残高・有効期限
    - チャージ専用ポイント残高・有効期限
    - 現在のスコア・ランク情報
- **エンコーディング**: サイトが `Shift-JIS` を使用しているため、取得時に `UTF-8` に変換して処理しています。
