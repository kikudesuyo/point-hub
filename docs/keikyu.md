# 京浜急行 (京急ポイント)

京急グループのポイントサービス「京急ポイント」の認証および取得方法について解説します。

## 認証方法

### 必要な情報
- **認証ID (ninsyo_id)**: 京急プレミアポイントの会員番号
- **パスワード (ninsyo_password)**: 登録時に設定したパスワード

### 手順
1. [京急プレミアポイント公式サイト](https://www.keikyu-point.jp/)で会員登録を行います。
2. 会員番号とパスワードを環境変数 `KEIKYU_LOGIN_ID` と `KEIKYU_PASSWORD` に設定します。

## ポイント取得の仕組み

### 技術的詳細
- **認証**:
    1. ログインページから `mg_token` (CSRF対策トークン) を取得します。
    2. 会員番号、パスワード、`mg_token` を含めて `https://kqpoint-portal.keikyu-point.jp/mypage/auth/login` へ `POST` リクエストを送信します。
    3. `User-Agent` を一般的なブラウザのものに偽装し、`Referer` や `Origin` ヘッダーを適切に設定する必要があります。
- **データ取得**: マイページトップ (`/mypage/`) を `GET` し、HTMLをパースします。
- **取得項目**:
    - 利用可能ポイント
    - 期間限定ポイント
    - 失効予定情報
