# 相模鉄道 (相鉄ポイント)

相鉄グループのポイントサービス「相鉄ポイント」の認証および取得方法について解説します。

## 認証方法

### 必要な情報
- **ユーザーID (Email)**: 登録したメールアドレス
- **パスワード**: 登録時に設定したパスワード

### 手順
1. [相鉄ポイント公式サイト](https://www.sotetsu-point.jp/)で会員登録を行います。
2. 相鉄ポイントマイル（乗車ポイント）を貯める場合は、別途PASMOの登録が必要です。
3. メールアドレスとパスワードを環境変数 `SOTETSU_EMAIL` と `SOTETSU_PASSWORD` に設定します。

## ポイント取得の仕組み

### 技術的詳細
- **認証**: 
    1. `https://mypage.sotetsu-point.jp/PISM010_00` を `GET` し、ログインフォーム内の CSRFトークン (`jp.hitachisoft.message.TOKEN`) を取得します。
    2. 取得したトークンと共に `PISM010_01` へ `POST` リクエストを送信してログインします。
- **データ取得**: マイページ (`PISM020_00`) を `GET` し、HTMLをパースします。
- **取得項目**:
    - 相鉄ポイント残高
    - ポイント有効期限
    - 相鉄マイル残高
    - マイル有効期限
- **特徴**: ポイントとマイルが分かれているため、両方のデータを個別に抽出しています。
